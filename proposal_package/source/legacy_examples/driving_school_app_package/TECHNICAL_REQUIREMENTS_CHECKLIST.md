# Technical Requirements Checklist

## Product Clarifications
- Preferred framework: Flutter or React Native?
- Is the target MVP Android-only first, or Android+iOS simultaneously?
- Should students choose instructor manually or auto-assigned by rules?
- Are lesson packages prepaid or pay-per-session?

## Booking and Scheduling
- Slot duration (30/45/60 minutes)?
- Reschedule/cancellation policy windows?
- Instructor overlap prevention rules?
- Holiday/leave handling logic?

## Student Progress Tracking
- What metrics define progress? (completed lessons, skills checklist, test readiness)
- Need assessment forms per lesson?
- Need downloadable progress report?

## Instructor Operations
- Required fields per session note?
- Attendance proof needed (OTP, geotag, signature)?
- Instructor payout reporting needed?

## Admin and Reporting
- Required dashboard KPIs?
- Export formats (CSV/PDF)?
- User role matrix (admin/manager/instructor/student)?

## Security and Compliance
- Authentication method (email, OTP, social)?
- Data retention policy?
- Privacy requirements by region?

## Release and Support
- App Store and Play Store publishing support needed?
- Post-launch support duration?
- Contract-to-hire role expectation and weekly availability?
